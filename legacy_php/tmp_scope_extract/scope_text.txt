Welfare contributions
Levies
Voluntary donations
Special fundraising
Payment tracking
Arrears
Receipts

Financial reports
1. Main Users/Roles
Administrator
Overall system manager.
Will
:
Manage users
Register members
Create dues/levies
Configure system settings
View all reports
Approve/void payments
Manage backups
Treasurer / Finance Officer
Handles money-related activities.
Will
:
Record payments
Verify payments
Generate receipts
View financial reports
Track arrears
Reconcile bank/
MoMo
payments
Secretary
Handles member records.
Will
:
Register and update members
View payment status
Generate member reports
Send notices
Auditor / Chairman
Read-only oversight role.
Can:
View reports
View audit logs
View collections
Check arrears
Export reports
Member
Optional portal user.
Will
:
Login
View dues
View arrears
Make payment
Download receipts
View payment history
or
Statement
2. Core Modules
A. Member Management
Capture:
Member ID
(Can be auto-generated or manually added)
Full name
Phone number
Email
Gender
Date of birth
Address
Occupation
Branch/group/unit, if applicable
Date joined
Status
Next of kin
Passport photo
(can be uploaded or taken with phone camera)
B. Groups / Branches / Zones
This only applies if the association has sub-groups.
Examples:
Branches
Zones
Departments
Units
Classes
Chapters
Electoral areas
Local groups
If the association is small, this can simply be disabled.
